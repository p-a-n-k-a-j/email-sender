# Email Sender

A simple bulk email sending web application built using Java Servlets, JSP, Bootstrap, and JavaScript.

The application allows users to send the same email message to multiple recipients at once by entering email addresses separated by commas. It is designed to reduce repetitive manual work when sending common announcements, greetings, notifications, or informational emails.

### Features

* Send emails to multiple recipients in a single request
* Simple and user-friendly interface
* Comma-separated email input support
* Custom subject and message content
* No user registration or login required
* Lightweight and easy to use
* Responsive UI built with Bootstrap

## Use Cases

This application can be useful in scenarios such as:

* Sending festival greetings (Diwali, New Year, etc.)
* Sharing announcements with multiple recipients
* Sending notifications to a group of users
* Educational or organizational communication
* Reducing repetitive email sending tasks

## How It Works

1. Enter one or more email addresses separated by commas.
2. Enter the email subject.
3. Write the message content.
4. Submit the form.
5. The application sends the same email to all provided recipients.

## Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/p-a-n-k-a-j/email-sender.git
```

### Navigate to Project Directory

```bash
cd email-sender
```

### Import into IDE

Import the project into your preferred IDE:

* IntelliJ IDEA
* Eclipse
* NetBeans

### Configure Email Credentials

Update the sender email configuration in the project before running the application.

### Deploy on Tomcat

1. Configure Apache Tomcat.
2. Build and deploy the project.
3. Start the server.
4. Open the application in your browser.

Example:

```text
http://localhost:8080/email-sender
```
